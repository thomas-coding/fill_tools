# 盛丹的小工具 EXE 使用说明（V0.2）

## 产物

- 可执行文件：`dist/盛丹的小工具.exe`

## 支持的 Excel 格式

- 直接支持：`.xlsx`, `.xlsm`, `.xltx`, `.xltm`
- 尽量支持：`.xls`, `.xlsb`, `.excel`

说明：
- 对 `.xls/.xlsb/.excel`，程序会尝试走 Excel COM 兼容路径；若环境缺少对应能力会给出提示。

## 模板逻辑（巡检模板）

- 每条记录为“标签行 + 内容行”两行结构。
- 巡检填报1映射：
  - 问题地址 <- 问题道路
  - 问题路段 <- 具体位置
  - 截止时间 <- 截止时间
  - 问题描述 <- 问题表述
  - 上传照片 <- 巡视问题
- 巡检填报2映射：
  - 处置路段 <- 问题道路
  - 整改描述 <- 整改描述
  - 上传照片 <- 处置情况（兼容“处理情况”表头）
- 图片提取兼容：标准 Excel 嵌图、WPS `DISPIMG/cellimages`、以及同目录 `wechat_extracted_images` 兜底。

## 运行流程

1. 启动 `盛丹的小工具.exe`
2. 在 UI 中选择功能（`巡检填报1` 或 `巡检填报2`）
3. 选择 Excel 文件
4. 点击“运行”
5. 在小程序中操作：
   - `F5`：按当前功能填文本（填报1填4项，填报2填2项）
   - `F12`：在文件框自动选图
   - `F10`：提交后标记已填并即时回写 Excel
6. 关闭程序窗口即停止热键和后台逻辑（关闭时不做回写）

## 状态规则

- 程序启动时按当前功能检查独立状态列：
  - 填报1：`截止时间`后第1/2列（状态/提交时间）
  - 填报2：`截止时间`后第3/4列（状态/提交时间）
- 如果手工删除“已填”，下次启动会当作未填写。
- 仅 `F10` 触发回写；直接关闭程序不会回写最后状态。

## 测试结果

- 单元测试：`python -m unittest discover -s tests -v` 通过
- 打包运行冒烟：`盛丹的小工具.exe` 可启动，进程可正常结束
