# 微信小程序填报助手（按你的 1.xlsx）

现在默认读取当前目录里的 `1.xlsx`。

如果你文件名是 `1.excel`，脚本会自动镜像成 `1.xlsx` 再处理。

## 数据映射（已按你要求）

- 问题地址 <- 表格列 `问题道路`
- 问题路段 <- 表格列 `具体位置`
- 截止时间 <- 表格列 `截止时间`
- 问题描述 <- 表格列 `问题表述`
- 上传照片 <- 表格列 `巡视问题`（优先读单元格路径；若是嵌入图片会自动导出到 `wechat_extracted_images`）

说明：
- 你的表是“标签行 + 内容行”成对出现（第1行为标签，第2行为内容；第3行为标签，第4行为内容），脚本已按这个结构解析。
- 问题类别、处置方式不再自动处理，改为你手动选择。

## 如何启动

- 双击 `run_wechat_helper.cmd`
- 启动时会自动生成：
  - `wechat_form_data.tsv`（填报数据）
  - `wechat_form_progress.tsv`（进度）
- 退出助手后会把进度回写到源 Excel：
- 按 `F10` 标记时会立即回写到源 Excel：
  - 若是 `1.xlsx` 这类巡检表：写到第16/17列（填报状态/提交时间）
  - 若是普通模板：写到第8/9列

进度判定规则：
- 程序启动时以 Excel 状态列为准。
- 如果你手工把“已填”删除，该条会被当成未填写。

## 每条怎么填

1. 在小程序新建页，先点到“问题地址”输入框。
2. 按一次 `F5`：自动按 Tab 顺序填好 4 项文本（地址/路段/截止/描述）。
3. 你手动选择“问题类别”。
4. 点“上传照片 +”，弹出文件框后按 `F12` 自动选图。
5. 你手动选择“处置方式”。
6. 点“确定”提交，然后按 `F10` 标记完成并跳下一条。

## 热键

- `F5` 一键填4个文本项
- `F8` 仅粘贴问题描述（手动点到“问题描述”输入框后按）
- `F12` 文件选择框自动填入照片路径
- `F10` 标记当前记录已填并跳下一条
- `F9` 下一条待填
- `F6` 上一条待填
- `F3` 描述前 Tab 跳过数 +1
- `F2` 描述前 Tab 跳过数 -1
- `F11` 打开当前记录照片
- `F4` 查看当前记录全部值
- `F1` 保存进度并退出
- `Ctrl+Esc` 退出

## 常见问题

- 打开 Excel 提示只读：运行 `unlock_excel_readwrite.cmd`，选 `y` 清理锁。
- `F5` 后“问题描述”没填到：先按 `F3` 增加跳过数，再重试 `F5`；或手动点描述框按 `F8`。
- 当前默认按你的页面设置：截止时间后只按 1 次 Tab 就到问题描述（额外跳过数=0）。
- 图片没选上：先确认你已点击“上传照片 +”打开系统文件框，再按 `F12`。
